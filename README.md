# Self-driving Car

## 국문
자율주행자동차 프로젝트입니다.

파이카메라로 캡쳐한 이미지의 하얀 길을 분석해 좌우로 회전하는 자동차입니다.

## 영문
It's a self-driving car project.

It turns left / right by detecting white lane in image that captured by PICamera.
